sacredx
===============================

version number: 0.1.0
author: Andrew Hundt

Overview
--------

run reproducible experiments in xonsh by integrating with https://github.com/IDSIA/sacred

Installation / Usage
--------------------

To install use pip:

    $ xip install xontrib-sacredx


Or clone the repo:

    $ git clone https://github.com/ahundt/sacredx.git
    $ xip install ./sacredx
